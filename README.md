Bask - Is a horror storyboard game built using Ren.py! Basic story of bask is it opens to a dark, dystopian type mansion located near a graveyard. There have been mysterious murders happening over the past few days between 5 friends living in the house. Here’s the twist- One of the friends is cursed spirit and if the four others simply escape and don’t find the cure,they get infected and eventually die.

In our game you are placed in the feet of the main character, Kage. Through him the user is able to make a series of decisions to continue the game. Focusing on mutiple choice and transitions.
